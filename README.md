# Customer Segmentation & Behavior Analysis (Excel)

## Overview
This project presents an end-to-end analysis of customer purchase data using Microsoft Excel to explore customer behavior, identify high-value customers, and segment customers based on spending and purchase frequency.
The insights from this analysis support improved customer retention strategies and revenue growth.

## Key Business Questions
- Who are the top-value customers?
- How frequently do customers make purchases?
- Which customers are loyal versus one-time buyers?
- Which customer segments generate the highest revenue?

## Dataset
The dataset consists of customer transaction records with the following fields:
- Customer ID
- Order ID
- Order Date
- Product
- Quantity
- Unit Price
- Total Amount

The data required cleaning and preparation before analysis.

## Data Preparation
Data cleaning and preparation were carried out in Excel and included:
- Structuring raw data into Excel tables
- Cleaning text fields and standardizing date formats
- Handling missing values and recalculating sales figures
- Creating helper columns for time-based analysis
- Validating and correcting data inconsistencies

## Analysis & Segmentation
- Built a customer-level summary table
- Calculated key metrics:
  - Total Spend
  - Purchase Frequency
  - Average Order Value
- Segmented customers by:
  - Value (High, Medium, Low)
  - Behavior (Loyal, Occasional, One-Time)
- Used Pivot Tables to analyze customer distribution and revenue contribution

## Visualizations & Dashboard
- Bar chart showing customers per segment
- Pie chart illustrating segment distribution
- Column chart displaying revenue by segment

An interactive Excel dashboard was created to highlight insights and trends.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel formulas (SUMIFS, COUNTIFS, IF)
- Charts and dashboards
